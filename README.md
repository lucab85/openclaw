# OpenClaw Starter Pack (Course Assets)

## What this is
A minimal set of memory + guardrails + one skill template used in the course.

## Folder guide
- memory/about_me.md -> your stable context and preferences
- memory/operating_rules.md -> safety and tool boundaries
- skills/run_daily_brief.skill.md -> a reusable automation pattern
- demos/ -> prompts + example output

## First run
1) Start OpenClaw (local Docker or VM)
2) Load memory files
3) Add the skill
4) Run: "run_daily_brief for today (draft only)"
